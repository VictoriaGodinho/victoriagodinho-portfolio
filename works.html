My Works

Project 1: Autobiographical Game

This is an autobiographical game where the character navigates through mazes reflecting my journey moving across countries.

Project 2: Data Visualization

This data portrait tracks the duration of my showers over a week. Each circle represents a shower, with lighter colors and slower pulsations for longer showers, and darker colors with faster pulsations for shorter ones. The design, featuring a shower background and interactive bubbles, aims to raise awareness about water waste.

Project 3: Optical Illusion

This project explores the hypnotic effect of the ripple illusion, where expanding and contracting circles create a rhythmic and mesmerizing visual flow. The repetitive symmetry is captivating, drawing the viewer into a meditative state.

Project 4: Experimental Clock

This experimental clock uses a jar to collect falling shapes that represent time: orange balls for seconds, yellow triangles for minutes, and red squares for hours. Inspired by how children learn to count, it reimagines time as a playful and visual experience.

Project 5: Exquisite Corpse

This character combines elements from a snowman, robot, and alien/octopus, capable of adapting to any environment.

Project 6: Face Generator

This generative art project creates unique faces with random colors and features influenced by user interaction.

Project 7: Charger

My first-ever coding project, this piece experiments with creating art using numbers. It taught me key concepts like push and pop functions and working with fill colors.
